\Frame_Factory

Contains frame decorating logic

This class is responsible for assigning the correct Frame_Decorator, Positioner, and Frame_Reflower objects to Frame objects. This is determined primarily by the Frame's display type, but also by the Frame's node's type (e.g. DomElement vs. #text)
