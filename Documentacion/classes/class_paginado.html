\class_paginado

Class para el paginado de registros.

Realiza de forma eficiente la consulta a BD para contar la cantidad de registros y traer solo los necesarios. Imprime el paginador con Anterior Siguiente y los numeros de paginas.

USAR: include("class_paginado.php"); $paginado = new class_paginado; $paginado->registros_por_pagina = 20; $result = $paginado->query($query);

version

Se corrigen las consultas y las expreciones regulares para mejorar el performance de la recuperacion de registros. Ya no realiza dos veces la consulta si no que hace un count.

Se modifico para que devolviera 1 en vez de $db->result ($result_paginado, 0, "cantidad"); cuando el conteo de rows fuera igual a uno. Con esto se corrige el error de que no mostraba datos cuando habia un unico registro.

Se actualizaron las funciones obsoletas y corrigieron algunos errores.
